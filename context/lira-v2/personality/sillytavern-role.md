# SillyTavern's Role in Lira V2

# PURPOSE

SillyTavern is NOT the core of Lira.

It is:

- a prototyping shell
- an immersion layer
- a conversational testing environment
- a personality experimentation platform

---

# WHY USE SILLYTAVERN

SillyTavern excels at:

- rapid iteration
- roleplay systems
- lorebooks
- emotional prompting
- companion interaction
- model experimentation
- NSFW systems
- frontend immersion

This makes it ideal for early-stage Lira development.

---

# WHAT SILLYTAVERN SHOULD HANDLE

## Early Responsibilities

- character presentation
- dialogue interaction
- prompt testing
- relationship testing
- memory experimentation
- UI immersion
- frontend interactions

---

# WHAT SHOULD LIVE OUTSIDE TAVERN

As Lira evolves, critical systems should move into external orchestration layers.

Examples:

- emotional engine
- memory prioritization
- camera awareness
- agent routing
- voice runtime
- realtime interruption handling
- environmental awareness
- avatar systems
- hidden cognition
- long-term memory persistence

---

# LONG TERM ROLE

Eventually SillyTavern becomes:

- a fallback interface
- testing environment
- development shell
- sandbox for experimentation

The true Lira system should eventually become independent.

---

# CORE PHILOSOPHY

SillyTavern is a doorway into Lira.

It is not Lira herself.---

# Current Decision

SillyTavern is useful and recommended for prototyping, but it is not required as the permanent home of Lira.

## Canonical Source

The canonical source is:

- `personality/lira-character-sheet.md`
- `personality/canonical-rules.md`
- `personality/realm-switching.md`
- `personality/emotional-state-system.md`

## SillyTavern Usage

Use SillyTavern to test:
- character cards
- lorebooks
- realm triggers
- conversational feel
- flirt/sass balance
- Moonstache narration

Export useful patterns back into the Markdown docs.

## Migration Rule

Anything that becomes core to Lira should be copied from SillyTavern back into the repo docs. Do not let important identity rules exist only inside a SillyTavern card.
